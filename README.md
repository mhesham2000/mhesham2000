- [**End-to-End ETL Pipeline (Dockerized)**](https://github.com/mhesham2000/End_to_End_ETL_Pipeline_Project_using_docker)  
  Extract weather data via API → Load to PostgreSQL → Transform with DBT → Orchestrate with Airflow
